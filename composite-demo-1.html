jQuery UI Composite Demo 1 - Flight Booking

A composite demo featuring multiple jQuery UI components/widgets in a "real world" scenario. This demo shows a potential flight picker.

Components/Widgets Used

  • Tabs (Tabs API)
  • Autocomplete (Autocomplete API)
  • Datepicker (Datepicker API)
  • Spinner (Spinner API)
  • Button (Button API)
  • Book A Flight
