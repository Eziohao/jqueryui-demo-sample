      • Demo Description

        This is a sample flight search/booking tool that makes use of the great functionality of multiple jQuery UI Widgets. Airport selection is made quick and easy for the user by using Autocomplete while choosing departure and return dates and time frames are incredibly user friendly thanks to Datepicker and Slider.

        The general UI is enhanced using the Spinner and Button widgets, while error handling is conveyed in a concise manner using the Dialog widget.
